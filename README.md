# ConfiguradorDePC

Este proyecto es un configurador de PC a medida desarrollado en Prolog.

## Descripción

El propósito de este proyecto es permitir la configuración personalizada de una computadora, utilizando lógica de Prolog para gestionar componentes y reglas de compatibilidad.
